This is a sudoku solver written in python that uses depth first search.

dependencies in requirements.txt. The only package required is tabulate

to use this tool you should call: python SudokuSolver.py [name of file]

It has been distributed with many test files to give you an idea on how to format the files.
an empty grid would look like:

000000000
000000000
000000000
000000000
000000000
000000000
000000000
000000000
000000000

These may be spaced out however you would like, but the puzzle MUST be in the first 9 lines of the file.
